# Danfoss Air CCM Roadmap

This document describes the planned development of the Danfoss Air CCM Home Assistant integration.

## Core Features

- [x] Config Flow
- [x] Native TCP communication
- [x] Fan Step control (1–10)
- [x] Current Supply Step
- [x] Current Extract Step
- [x] Basic Supply Step control
- [x] Basic Extract Step control
- [x] Outdoor Temperature
- [x] Supply Temperature
- [x] Extract Temperature
- [x] Exhaust Temperature
- [x] Relative Humidity
- [x] Native Bypass switch
- [x] Installer Mode
- [x] Persistent installer settings
- [x] Restore Installer Settings
